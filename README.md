# react+ts+weather+app

## 프로젝트 개요
React + TypeScript + Vite로 만든 날씨 앱

## 기능 목록
- 현재 위치의 날씨 자동으로 가져오기
- 도시 이름으로 날씨 검색
- 실시간 날씨 데이터 표시
- 날씨 항목별 상세 정보 표시
- 날씨 상태에 따라 아이콘 이미지 표시

## 기술 스택
- HTML
- CSS
- React
- TypeScript
- Vite
